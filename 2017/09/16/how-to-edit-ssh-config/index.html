~/.ssh/configを編集する

k4h4shi.com
2017-09-16
config, ssh, tips

~/.ssh/configはsshクライアントの設定ファイルです。
ssh接続先の情報を定義しておくことで、sshコマンドのオプションを省略できる。

sshコマンドのオプション

  • -i: 秘密鍵を指定
  • -p: ポート番号を指定
  • -l: ログインユーザ名を指定

.ssh/configの設定例

上記のように設定をすることで、ssh hogeとコマンドを実行するだけでサーバにログインできるようになります。
