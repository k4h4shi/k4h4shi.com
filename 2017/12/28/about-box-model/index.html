box model

box model は、HTML の要素を CSS でスタイリングする際の基本的なモデルになります。

box model は以下の構成要素を持ちます。

box model の構成要素

  • content: box, または HTML の要素の内容。
  • height: content の高さ
  • width: content の幅
  • border: content が入った box の外枠
  • padding: content と border の間のスペース
  • margin: box の外側のスペース
